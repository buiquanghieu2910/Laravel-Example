# Laravel-Example

## Authencation-Laravel-Example
- **Description:** Xác thực API sử dụng JWT (lib: tymon/jwt-auth)
- **Run CMD:**
  - Config file .env
  - composer update
  - php artisan cache:clear
  - php artisan config:cache
  - php artisan key:genernate
  - php artisan migrate
  - php artisan db:seender
  - php artisan db:seed
  - php artisan serve
